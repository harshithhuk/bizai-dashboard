# 🤖 BizAI — AI-Powered Business Intelligence Dashboard

**Author:** Harshith Pabbathi | MSc Data Science, Northumbria University  
**GitHub:** [harshithhuk](https://github.com/harshithhuk)

> **No API key required** — runs entirely in the browser using a rule-based NLP engine

---

## 📌 Overview

A natural language business intelligence dashboard where you type plain English queries and get instant data visualisations. Combines NLP intent detection, data analysis, and interactive charting in a single zero-dependency HTML file.

**Try asking:**
- *"show me sales by region"*
- *"monthly revenue trend"*
- *"top products this year"*
- *"profit margin by category"*
- *"show anomalies"*
- *"sales rep performance"*

---

## 🧠 Skills Demonstrated

| Skill | Implementation |
|-------|---------------|
| Natural Language Processing | Intent detection engine with keyword scoring |
| Data Visualisation | Line, bar, doughnut charts via Chart.js |
| Business Analytics | KPIs, regional analysis, cohort segmentation |
| Anomaly Detection | Statistical outlier flagging (mean ± std dev) |
| Data Storytelling | AI-generated insights per query |
| Frontend Engineering | Dark-mode dashboard, responsive layout |
| Prompt Engineering | Structured NLP intent classification |

---

## 📊 Dashboard Features

- **KPI Cards** — Revenue, Orders, AOV, Profit Margin with YoY comparison
- **Natural Language Query Bar** — type any business question
- **Dynamic Charts** — updates based on your query
- **8 Query Intents** — region, monthly, products, category, quarterly, sales reps, segments, anomaly
- **AI Insight Panel** — plain English summary of findings
- **Query History** — click to re-run previous queries
- **Regional Breakdown** — UK sales by geography
- **Leaderboard** — sales rep performance ranking
